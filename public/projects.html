Academic & Personal Projects

  1. Personal Resume – Website (Academic) February 2023

    • The features are a personal bio of myself, my full professional resume, a list of my projects that I've completed in the past few years, my contact information, my socials, and an admin area.

  2. Data Catalog – Web Application (Internship)September 2022– December 2022


    • Continued the development of a web application that creates a method for users to pick and choose data they wish to export, which will be used in their respective projects. This application was developed using Virtual Box, Linux/Ubuntu, Graph Databases, Python, Docker containers, Apache Airflow, Vue.js, neo4j and Hadoop cluster.
    • The project is divided in three components: a user interface, a backend which is the core of the application and a scheduler to fetch data selected from the user.
    • The web application permits the user to upload graph structures and select the nodes that describe the data the need, the data is then fetched from the file storage system.

  3. Resource Portal – Web Application (Internship) September 2022– December 2021


    • Developed a web application that allows users to form different groups (for different projects or different companies) to manage virtual machines for their respective projects. The application was built with Python 3, Django, JavaScript, MySQL and FreeIPA.
    • The features that have been implemented so far are a frontend with user input signup and login validation, a maintenance-mode for special uses and an IPA authentication.

  4. Sub-Dictionary Creator & Cell Phones Records (Academic) April 2020


    • Designed a program allowing an input file (type “.txt”) to be read and create a small dictionary of its contents, using File I/O and Array List. This sub-dictionary is the first part of a two-part assignment.
    • Applied concepts of linked lists and File I/O to create a program that reads cell phones records from a text file and store them into a list to make the information easily accessible.

  5. Flying Devices – Inheritance, Polymorphism and Abstract Concepts Practice (Academic)February 2017


    • Created a program which implements the basic concepts of inheritance as well as polymorphism.
    • The different types of planes are connected to their base class “Airplane” and the types of drones, to their base class “UAV”. These two base classes are then connected to an abstract class of “Flying devices”, which allows the classes to access each other’s variables and methods (through polymorphism).
